## INSTALLATION
# Termux
```bash
> Download Termux
> Buka APK Termux nya
> apt update && apt upgrade
> git clone https://github.com/ben-dev2609/whatsapp-termux-bot.git
> cd whatsapp-termux-bot
> pkg install wget
> pkg install ffmpeg
> pkg install nodejs
> npm i -g cwebp
> npm i -g ytdl
> npm i
> npm i got
> node index.js
```
